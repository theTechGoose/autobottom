# Developer Role Spec

A new user type that sits above organizations at
the platform level.

---

## 1. Developer

- Has their own team that spans multiple orgs
- Can impersonate admins at top level and
  everything down the tree
- Creates providers (integration definitions)
- Default dashboard with app-wide
  stability/performance metrics
- Builds and publishes marketplace items

---

## 2. Hierarchy

```
Developer (platform level)
  Admin (org level)
    Supervisor / Judge / Analyst / Agent (team level)
```
